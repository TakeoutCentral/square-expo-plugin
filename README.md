# square-expo-plugin

This is an Expo plugin for the [Square In-App Payments SDK](https://developer.squareup.com/docs/in-app-payments-sdk/overview) that allows you to accept payments in your React Native app.

## Add the package to your npm dependencies

```
npm install --dev square-expo-plugin
```
